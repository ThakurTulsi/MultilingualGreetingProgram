# MultilingualGreetingProgram

## Overview
This Python script, **Voice Assistant Greetings**, is a small application that allows users to choose a voice assistant (either Male or Female) and receive personalized greetings in multiple languages.

## Prerequisites
Before running the script, ensure you have the following libraries installed:
- [SpeechRecognition](https://pypi.org/project/SpeechRecognition/) - Install using `pip3 install SpeechRecognition`
- [pyttsx3](https://pypi.org/project/pyttsx3/) - Install using `pip3 install pyttsx3`

## How to Use
1. Run the script.
2. Select a voice assistant (1 for Male, 2 for Female).
3. Choose a greeting option:
   - 1: English - "Hello"
   - 2: Hindi - "Namaste"
   - 3: French - "Bonjour!"
   - 4: Enter a personalized greeting

## Features
- The script uses the SpeechRecognition library to capture user input.
- It provides a menu for greeting selections.
- Users can choose between a male or female voice assistant.
- The script supports greetings in English, Hindi, and French.
- Users can input their personalized greetings.
- Voice output is generated using the pyttsx3 library.

## Note
- In this code, we use male and female voice options. If you encounter any issues with voice selection, you may need to adjust the voice preferences for your system.
